The Unjournal vs Traditional Journal Process

Traditional economics journals average 24+ months from submission to publication. The Unjournal aims for weeks, with transparent, paid evaluation.

Key Differences

Time — 24+ months → 7-10 weeks
Transparency — Secret reviews → Public evaluations
Compensation — Unpaid → $200-400 per evaluation
Output — Binary accept/reject → Multi-dimensional ratings
Format — PDF only → Dynamic documents welcome
Citation — Journal prestige → Direct quality metrics
