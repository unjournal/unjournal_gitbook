# PQ: WELLBY Reliability

Questions about whether the linear WELLBY is a reliable measure for comparing interventions.

---

## Questions in This Category

### WELL_01
**How reliable is the WELLBY relative to other wellbeing measures for comparing interventions?**
- Type: Probability estimate
- Key consideration: Does the simplicity justify potential measurement error?

### WELL_02
**Given available data, how should funders measure wellbeing impact?**
- Type: Open-ended
- Key consideration: Practical constraints vs theoretical ideals

### WELL_03
**What measures should charities/NGOs/RCTs collect for impact analysis?**
- Type: Open-ended
- Key consideration: Survey burden vs information value

### WELL_04
**Is it better to use a single WELLBY measure or measure dimensions separately?**
- Type: Categorical (single_wellby / separate_measures / depends_on_context)
- Key consideration: Aggregation vs granularity

### WELL_07
**What is lost by using linear WELLBY instead of more complex measures?**
- Type: Open-ended
- Key consideration: Quantifying the trade-off

### WELL_08
**Should the metric be based on life satisfaction or experience measures?**
- Type: Categorical (life_satisfaction / experience / both / unsure)
- Key consideration: Evaluative vs experiential wellbeing

### WELL_09
**If relying on Cantril ladder, how best to convert to welfare metric?**
- Type: Open-ended
- Key consideration: Scale anchoring and interpretation
