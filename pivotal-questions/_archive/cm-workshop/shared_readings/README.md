# Cultivated Meat Workshop Reading Collection

Pre-readings and reference materials for the Unjournal Cultivated Meat Workshop.

PDFs are collected in a [shared Dropbox folder](https://www.dropbox.com/scl/fo/abrcf9328cs08tee08hw1/AFFDPrJzNd7gDh-iojy4W3E?rlkey=wr6fkz8qbw0a6bay9osfjnirf&dl=0), organized as follows. The folder can be downloaded and uploaded to NotebookLM for AI-assisted comparison across papers.

## Folder Structure

```
cm_research/
├── (15 focal papers at root)
├── gfi-reports/        (8 GFI industry reports)
├── media-and-inputs/   (4 cell culture media papers)
└── _tracking/          (stale inventory CSVs)
```

## Essential Readings (root)

The core disagreement the workshop is built around.

| File | Citation | Why it matters |
|------|----------|----------------|
| `Pasitka_2024_Nature_Food_Empirical_TEA.pdf` | Pasitka et al. (2024). Nature Food. | The optimistic TEA; headline claim ~$6/lb. |
| `Humbird_2021_Scale_Up_Economics.pdf` | Humbird (2021). Open Philanthropy. | Foundational skeptical analysis; $17–23/lb. |
| `Goodwin_2024_Scoping_Review_TEAs.pdf` | Goodwin et al. (2024). Nature Food. | Best overview of where TEAs agree and disagree. |
| `CE_Delft_2021_TEA_Cultivated_Meat.pdf` | CE Delft (2021). TEA report. | Widely cited midpoint/reference case. |
| `Risner_2021_Preliminary_TEA.pdf` | Risner et al. (2021). Foods/MDPI. | Early TEA establishing cost framework. |
