### Features to Leverage

- **Breakout rooms**: Enabled with full features. Only the host/co-host can create breakout rooms (participants cannot self-create). To enable spontaneous collaboration: create pre-named rooms ("Side Discussion A", "Side Discussion B") and let participants self-select into them.
- **Cloud recording**: Auto-record enabled. Single continuous recording — Zoom doesn't auto-split by segment. Use "Pause/Resume" recording between segments to create natural chapter breaks. AI Companion can add chapters/highlights after processing.
- **Chat**: Use for procedural announcements only. Pin a message at start: "For substantive discussion, please use the Google Doc: [link]". Repeat at each segment transition.
- **Automated captions**: Already enabled, multi-language.
- **Full transcript**: Will be saved automatically with cloud recording.

## Reference: How to Change Zoom Settings

**User-level settings** (per-meeting defaults): [zoom.us/profile/setting](https://zoom.us/profile/setting)
**Account-level settings** (admin, unlocks features): [zoom.us/account/setting](https://zoom.us/account/setting)

Some features (like AI Companion, Co-host) must be enabled at the account level first before they appear in user settings or per-meeting options.
